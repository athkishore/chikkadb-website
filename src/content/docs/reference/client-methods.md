## Database Methods
### `db.createCollection`

Creates new collection in the database.

Parameters:

- `name`: Name of the new collection to be created

Currently options for collections are not supported.

### `db.dropDatabase`

Deletes the database.

---

### `db.getCollectionNames`

Returns the list of collections in the database.
